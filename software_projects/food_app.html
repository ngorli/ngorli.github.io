Recipe Sharing Social Media App

For my senior group project in Computer Science, my team and I developed NENA, a social media application designed for sharing and discovering recipes. The app was built using Firebase and React Native, providing a platform for users to express their culinary creativity. With NENA, users can upload and share recipes, view others' recipes, search for recipes, and connect with friends.

  • Overview
    • NENA aims to provide a dynamic and engaging environment for users to showcase their cooking skills and explore new recipes. The app allows users to:
      • Share Recipes: Upload and display their favorite recipes.
      • View Recipes: Browse recipes shared by others on their friends feed.
      • Search Recipes: Find specific recipes using search functionality.
      • Add Friends: Connect with other users and interact through the app.
  • Backend Development
    • In the backend, I focused on setting up the database structure in Firebase and implementing essential functions. The backend architecture includes:
      • User Collection: Each user has fields for date of birth (DOB), cooking level (selectable from predefined levels), dietary restrictions, email address, first and last names, friends list, friend requests, pending requests, passwords, user IDs, and profile picture URL.
      • Recipes Collection: Each recipe entry contains lists for ingredients and instructions, public visibility status, a description, a picture, a title, a creation timestamp, and the associated user ID.
    • Key tasks I undertook included:
    • Designing the Database Schema: Creating a hierarchical structure for users and recipes.
    • Implementing Backend Functions: Developing functions to handle friend-related processes (sending, accepting, and denying friend requests) and user management (creation and authentication).
  • Frontend Development
    • I also did work on the frontend of the application. Below are all the screens I worked on in this application
      • Home Screen The landing page for logged-in users. This screen displays the number of friends and recipes, along with images of the user's recipes. Users can click on a recipe to view its details or click on the number of friends to navigate to the Friends Screen.
  • Friend Screen Lists all the user's friends and allows users to search for other users to send friend requests. Users can also accept or deny incoming friend requests. screen.
  • Feed Screen Shows a scrollable feed of recipes uploaded by friends. It displays the recipe title, description, user, and image.
  • Search Screen Enables users to search for recipes by title or description. The search results include the recipe title, description, user, and image. Users can click on a recipe to view its detailed page.
  • Recipe Screen Provides detailed information about a selected recipe, including ingredients, a picture, and step-by-step instructions.
